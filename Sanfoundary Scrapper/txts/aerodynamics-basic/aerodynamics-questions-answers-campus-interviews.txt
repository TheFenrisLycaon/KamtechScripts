1. The flow in which streamlines are directed away from the origin is called as __________
a) sink flow
b) doublet flow
c) source flow
d) source-sink flow
c
2. The opposite case of the source flow is ___________
a) sink flow
b) doublet flow
c) source flow
d) source-sink flow
a
3. The origin is called as _________
a) singular point
b) multiple point
c) sink point
d) source point
a
4. In the source flow, the tangential velocity component is  _________
a) 0
b) 1
c) not defined
d) infinity
a
