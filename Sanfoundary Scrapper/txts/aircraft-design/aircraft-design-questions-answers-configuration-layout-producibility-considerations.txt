1. Aircraft production cost is related to _____________
a) weight, size, material, etc
b) weight only
c) size only
d) material only
a
2. Aircraft production cost can be reduced by using _____________
a) flat wrap surfaces
b) non-flat surface
c) more forging processes
d) welding only on non-flat surfaces
a
3. _____________ are one of the most expensive type of structure in general.
a) Forging
b) Lifting only
c) Non-lifting only
d) Only high speed bodies
a
4. To simplify routing requirements we can provide _____________
a) routing tunnels
b) structural break
c) reducing lift at cruise
d) increasing cruise weight
a
5. Following diagram represents _____________
a) typical routing tunnel
b) typical non-routing
c) non routing member
d) avionics only
a
6. Careful placement of the internal components can reduce routing cost.
a) True
b) False
a
7. Following diagram represents ____
a) poor production break design
b) preferred production break design
c) best production brake design
d) structural cut out as per production cost concern
a
8. To reduce production cost there should be _____________
a) commonality of parts
b) more forging
c) increment in forge and labour requirements
d) decrease commonality of parts
a
9. Which of the following is correct?
a) If possible, designer should omit unnecessary part to reduce cost
b) Lift is always same as weight of the aircraft
c) Producibility concern should not be used at all
d) Only forging process will affect cost
a
10. Tight internal packaging will ease the process of routing.
a) True
b) False
b
