1. Which of these is not necessary for the description of an automobile?
a) Type
b) Capacity
c) Colour
d) Model

2. Which of these is necessary for the description of an automobile?
a) Make
b) Model
c) Capacity
d) All of the mentioned

a) Ultra-compact car
b) Sub-compact car
