1. Echocardiography is essentially ________
a) ultrasound of the heart
b) echoing sound of the heart
c) another name for a treadmill test
d) recording of heart sounds
a
2. The probe used for imaging the heart is ________
a) Linear Probe
b) Curvilinear Probe
c) Phase Array Probe
d) Endocavitary Probe
c
3. Which mode is used for cardiac imaging?
a) A – Mode
b) B – Mode
c) M – Mode
d) 3D – Mode
c
4. Continuous Wave (CW), Pulsed Wave (PW) and Color Flow (CF) are terms used for which kind of ultrasound?
a)  3D ultrasound
b) Dynamic Ultrasound
c) Doppler Ultrasound
d) Advanced Ultrasound
c
5. Echocardiography can be used for even _________
a) remove embolus of the heart
b) correctly locate the blockages in the arteries and veins
c) assess the baby’s heart, anatomy and function
d) measure heartbeat
c
6. Which kind of echocardiography may be recommended to diagnose coronary heart disease?
a) Thoracic Echocardiography
b) Stress Echocardiography
c) Transesophageal Echocardiography
d) 3D Echocardiography
b
7. Which of the following ultrasound techniques gives the best Ultrasound Image of the heart?
a) 2D phased array
b) Linear phased array
c) Transesophageal
d) Curvilinear array
c
8. The following is an image of a curvilinear probe. What is best imaged with a curvilinear probe?
a) superficial structures and vessels
b)  transabdominal imaging
c) cardiac imaging
d) transvaginal and transrectal imaging
b
