2. The set of all strings over ∑ = {a,b} in which all strings having bbbb as substring is?
a) (a+b)* bbbb (a+b)*

a) ab* + ba*
b) ab*ba*
c) a*b + b*a
d) None of the mentioned
a

a) Recursive language
b) Context free language
c) Regular language
d) All of the mentioned

b) a*bbb

a) R = QP*
b) R = P*Q
c) R = RP
d) None of the mentioned
a

a) L1 + L2
b) L1L2
c) L1

a) a/b
b) (a/b)*
