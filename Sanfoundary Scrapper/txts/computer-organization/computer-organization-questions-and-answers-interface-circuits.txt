1. ______ serves as an intermediary between the device and the BUSes.
a) Interface circuits
b) Device drivers
c) Buffers
d) None of the mentioned
a
2. The side of the interface circuits, that has the data path and the control signals to transfer data between interface and device is _____
a) BUS side
b) Port side
c) Hardwell side
d) Software side
b
3. What is the interface circuit?
a) Helps in installing of the software driver for the device
b) Houses the buffer that helps in data transfer
c) Helps in the decoding of the address on the address BUs
d) None of the mentioned
c
4. The conversion from parallel to serial data transmission and vice versa takes place inside the interface circuits.
a) True
b) False
a
5. The parallel mode of communication is not suitable for long devices because of ______
a) Timing skew
b) Memory access delay
c) Latency
d) None of the mentioned
a
6. The Interface circuits generate the appropriate timing signals required by the BUS control scheme.
a) True
b) False
a
7. The status flags required for data transfer is present in _____
a) Device
b) Device driver
c) Interface circuit
d) None of the mentioned
c
8. User programmable terminals that combine VDT hardware with built-in microprocessor is _____
a) KIPs
b) Pc
c) Mainframe
d) Intelligent terminals
d
9. Which most popular input device is used today for interactive processing and for the one line entry of data for batch processing?
a) Mouse
b) Magnetic disk
c) Visual display terminal
d) Card punch
a
10. The use of spooler programs or _______ Hardware allows PC operators to do the processing work at the same time a printing operation is in progress.
a) Registers
b) Memory
c) Buffer
d) CPU
c
