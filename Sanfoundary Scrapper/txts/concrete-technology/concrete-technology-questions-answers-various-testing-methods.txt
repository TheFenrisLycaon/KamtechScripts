1. Identify the type of destructive testing.
a) Radiographic test
b) Dye penetrant test
c) Creep test
d) Visual testing
c
2. Which test can be performed without skilled labor?
a) Probe test
b) Bend liquid test
c) Dye penetrant test
d) Torsion test
c
3. Compressive strength of hardened concrete is done by _______
a) Cube test
b) Tensile splitting test
c) Concrete core test
d) Flexure test
a
4. In the soundness test a specimen of hardened cement paste is _______ for a fixed time.
a) Freeze
b) Dry
c) Boiled
d) Dipped in water
c
5. On cooling below 1250oC, C3S decomposes ______
a) Fast
b) Slowly
c) Never
d) Depends on the conditions
b
6. The loss in weight should not exceed ______ percent when tested with sodium sulphate and ______ percent with magnesium sulphate solution.
a) 12, 18
b) 18, 12
c) 10, 15
d) 15, 10
a
