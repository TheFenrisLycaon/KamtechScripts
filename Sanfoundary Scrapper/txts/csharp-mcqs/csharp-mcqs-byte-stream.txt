1. Which of these classes is used to read and write bytes in a file?
a) FileReader
b) FileWriter
c) FileInputStream

a) int
b) float
c) byte
d) none of the mentioned

a) InputStream
b) Reader
c) Writer

a) BufferedStream
b) FileStream
c) MemoryStream

a) BufferedStream
b) FileStream

b) long Position

b) FileMode.Create
