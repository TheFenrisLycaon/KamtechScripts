1. In the drawing sheet which side of the borderlines more space is kept for the purpose of filing or binding when possible?
a) Left-hand side
b) Topside
c) Right-hand side
d) Bottom side
a
2. The trimmed size of the sheet designation A2 is _______ mm.
a) 841 X 1189
b) 594 X 841
c) 420 X 594
d) 450 X 625
c
3. For the sheets A0 and A1, what is the width of the border?
a) 10mm
b) 20mm
c) 15mm
d) 5mm
b
4. For the sheets A2, A3, A4, and A5 what is the width of the border?
a) 10mm
b) 20mm
c) 15mm
d) 5mm
a
5. Which of the following is used for positioning of the drawing sheet on the drawing board?
a) The grid system
b) Borders
c) Orientation marks
d) Folding marks
c

6. The grid reference system is also called a zone system.
a) True
b) False
a
7. Grids along the length are referred by which of the following types?
a) A, B, C, D,…
b) a, b, c, d,…
c) 1, 2, 3, 4,..
d) i, ii, iii, iv,…
c
8. Grids along the width are referred by which of the following types?
a) A, B, C, D,…
b) a, b, c, d,…
c) 1, 2, 3, 4,..
d) i, ii, iii, iv,…
a
9.  In the drawing sheet which side is given space for the title block?
a) Left-hand side
b) Topside
c) Right-hand side
d) Bottom side
c
10. Names of the persons who have designed and drawn the drawing are mentioned in ________
a) Borders
b) Margins
c) Title box
d) Grid system
c
11. After folding the sheet what is visible on the top?
a) Drawing
b) Dimensions
c) Title box
d) Alterations
c
12. Nine folds are allowed to which of the sheet designation when they are filed?
a) A0
b) A1
c) A2
d) A3
a
13. Seven folds are allowed to which of the sheet designation when they are preserved in the cabinet?
a) A0
b) A1
c) A2
d) A3
a
14. When a drawing sheet is stored in the cabinet, which of the following should be the final dimension of the fold? (In mm)
a) 297 X 190
b) 297 X 210
c) 220 X 190
d) 220 X 210
b
15. When a drawing sheet is filed or bound, which of the following should be the final dimension of the fold? (In mm)
a) 297 X 190
b) 297 X 210
