1. In passive filters amplitude-frequency chara of pass band region have_______________
a) Positive slope
b) Negative slope
c) Zero slope
d) None of the mentioned
c
2. Passive filters have infinite attenuation in stop band region.
a) True
b) False
b
3. Which of the following causes noise in passive filters?
a) Capacitor
b) Resistor
c) Inductor
d) None of the mentioned
c
4. Which of the filters are lower order filters?
a) Butterworth filter
b) Chebychev filter
c) Bessel filter
d) None of the mentioned
d
5. Which of the following have pass band of low frequency range?
a) High pass filter
b) Band pass filter
c) Low pass filter
d) Band stop filter
c
6. Which of the following is known as equiripple filters?
a) Chebychev filter
b) Bessel filter
c) Butterworth filter
d) None of the mentioned
a
7. Sharpness of cutoff in Butterworth filter __________ with increasing order.
a) Increases
b) Decreases
c) Nothing happens
d) None of the mentioned
a
8. Which of the following filter is used where the phase is of higher priority?
a) Chebychev filter
b) Butterworth filter
c) Bessel filter
d) None of the mentioned
c
9. Biquad network is a ___________ network.
a) 2 terminal
b) 4 terminal
c) 3 terminal
d) 5 terminal
b

a) Low pass filter
b) High pass filter
c) All pass filter
d) All of the mentioned
