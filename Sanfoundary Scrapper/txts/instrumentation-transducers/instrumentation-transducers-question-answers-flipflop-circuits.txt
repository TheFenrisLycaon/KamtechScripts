1. One flip-flop unit can store __________ of data.
a) One nibble
b) One bit
c) One byte
d) One kilo-bit
b
2. SR flip-flop stands for ______________
a) State-reset
b) Set-reset
c) Set-recovery
d) Set-reset
d
3. Which of the following represents D flip flop?
a) Data flip flop
b) Delay flip flop
c) Direct flip flop
d) Decode flip flop
b
4. SR flip flop have undefined state when______________
a) Both inputs are low
b) Both inputs are high
c) One input is low
d) None of the mentioned
b
5. Latch is a circuit which operates according to a clock pulse.
a) True
b) False
b
6. Race around condition can be rectified by ___________
a) SR flip flop
b) JK flip flop
c) JK master slave flip flop
d) D flip flop
c
7. Which of the following is an application of flip-flop?
a) Shift register
b) Latch
c) Gates
d) All of the mentioned
a
8. If SET pin is active, output becomes ______________
a) High
b) Low
c) Undefined
d) None of the mentioned
a
9. In SR flip flop if both inputs are low, output becomes _______________
a) High
b) Low
c) No change
d) Undefined
c
10. Which of the following is not a sequential circuit?
a) Flip flop
b) Counter
c) Logical gate
d) Shift register
c
