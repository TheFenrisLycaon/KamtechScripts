1. Which of the following is correct for data transfer in shift register?
a) Serial manner
b) Parallel manner
c) Both serial and parallel
d) None of the mentioned
c
2. Which of the following is correct for serial mode shift register?
a) One bit per clock pulse
b) One byte per clock pulse
c) Two bit per clock cycle
d) Unpredictable
a
3. Which of the following flip flops can be used for making shift registers?
a) T flip flop
b) JK flip flop
c) SR flip flop
d) All of the mentioned
d
4. Which of the following is not a valid BCD?
a) 0110
b) 0111
c) 1000
d) 1010
d
5. Which of the following have high propagation delay?
a) Synchronous counter
b) Asynchronous counter
c) Both have the same propagation delay
d) Both have an unpredictable propagation delay
b
