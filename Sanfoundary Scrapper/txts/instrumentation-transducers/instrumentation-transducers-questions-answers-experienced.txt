1. Which of the following uses displacement to pressure conversion?
a) Flapper nozzle system
b) Gyroscope
c) Viscometer
d) None of the mentioned
a
2. Displacement to pressure systems are used for measuring ___________
a) Displacement
b) Velocity
c) Acceleration
d) Force
a
3. Which of the following represents the correct relation between pressure and displacement in flapper nozzle system?
a) Direct proportionality
b) Inverse proportionality
c) Equal
d) None of the mentioned
b
4. Which of the following represents normal value for Pout/Pin?
a) Less than 0.2
b) Between 0.2 and 0.75
c) Between 0.75 and 1
d) Greater than 1
c
5. Which of the following is correct for flapper nozzle system?
a) Used in hydraulic system only
b) Used in pneumatic system only
c) Used in both hydraulic and pneumatic system
d) None of the mentioned
c
6. Range of displacement of flapper required to change output pressure over the useful range is known as ___________
a) Cut-off displacement
b) Throttling range
c) Threshold value
d) Break value
b
7. Which of the condition is applicable for flapper nozzle system?
a) Measuring air should be free from dust
b) Measuring air must be of constant pressure
c) Measuring air should be of free from dust and of constant pressure
d) None of the mentioned
c
8. Which of the following represents condition for a volume of chamber?
a) As large as possible
b) Limited to 5 cc
c) AS small as possible
d) No limit
b
9. Which of the following represents ideal flapper nozzle system?
a) Ratio of output and input pressure 0.75 at displacement = 0
b) Ratio of output and input pressure 1 at displacement = 0
c) Ratio of output and input pressure 0.5 at displacement = 0
d) Ratio of output and input pressure 0 at displacement = 0
b
10. At which point of displacement pressure chara has maximum slope?
a) 0.14(di/dt)2
b) (di/dt)
c) 0.5(di/dt)2
d) None of the mentioned
a
