1. Which of the following are not suitable with electromagnetic flow meter?
a) Steady flowing liquid
b) Slurries
c) Liquid containing particulate matter
d) None of the mentioned
d
2. Output signal of electromagnetic flow meter inversely proportional to _________________
a) Magnetic flux density
b) Diameter of tube
c) Velocity of flow
d) None of the mentioned
d
3. Frequency of AC electromagnets are independent of magnetic flux density.
a) True
b) False
b
4. Feedback system in electromagnetic flow meter is applied to ______________
a) Overcome fluctuation of B
b) Reduce measuring time
c) Reduce excitation supply
d) None of the mentioned
a
5. Feedback system draws current from the main circuit.
a) True
b) False
b
6. Servo operated electromagnetic flow meter is _________________
a) Zero order system
b) First order system
c) Second order system
d) Third order system
c
7. Electromagnetic flow meter is based on ________________
a) Coulomb’s law
b) Law of electromagnetic induction
c) Kirchhoff law
d) Newton’s law
b
8. Which for the following represents condition for maximum emf?
a) Electrodes parallel to pipe length
b) Electrodes parallel to flow
c) Electrodes across flow
d) None of the mentioned
c
9. Which of the following device is used for measuring emf between electrodes?
a) Low impedance type voltmeter
b) High impedance voltmeter
c) No specification for voltmeter
d) None of the mentioned
b
10. Which of the following factors electromagnetic flow meters are sensitive to?
a) Temperature of fluid
b) Viscosity of fluid
c) Density of fluid
d) None of the mentioned
d
