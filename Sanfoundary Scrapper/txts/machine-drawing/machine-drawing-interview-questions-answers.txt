1. Dimensions given in a drawing are the scaled dimensions and not the actual dimensions of the object for scaled views.
a) True
b) False
b
2. As far as possible dimensions should be given in one unit, preferably in _______
a) Meters
b) Centimeters
c) Millimeters
d) Inches
c
3. Addition of numerals on the drawing defines the object for ______
a) Shape
b) Size
c) Projections
d) Tolerances
b
4. Size is given by ________ for linear distances, radii, diameters, angles, etc.
a) Dimensions
b) Measurements
c) Arrows
d) Projections
a
5. Which of the following is the method of preparing drawing?
a) Machine drawing
b) Production drawing
c) Assembly drawing
d) Finished drawing
d
6. Output of CAD is not _________
a) Printed with a printer if it is small up to A4 size
b) Saved in the magnetic memory of the computer
c) Drawn on the paper with special drawing apparatus
d) Printed with multiple plotters if it is large
c
7. While preparing a drawing with freehand sketching _______
a) Special apparatus are used to draw curves
b) Angles are drawn with a protractor
c) Straight lines are drawn with set square
d) Sketching is done without any aid of drawing apparatus
d
8. CAD is the method of preparing drawings, which is an abbreviation of ______
a) Computer Aided Drafting
b) Call a Design
c) Completely Automated Designing
d) Common Application Data
a
9. Which of the following statement is not about Computer Aided Drafting?
a) Quality of such drawing is excellent
b) Output is taken with the help of plotter or printer
c) Drawings are saved in the magnetic memory of the computer or hard disk
d) Letters are written using stencils for good appearance
d
10. In a finished drawing, the appearance of the drawing does not depend upon the skill of the operator.
a) True
b) False
b
11. Which of the following option is not expressing the freehand sketch method of preparing drawing?
a) Sketching is done with pencil and paper
b) Lines may not be the exact length in such drawings
c) It is good in learning process
d) In sketching, circles are drawn with compass
d
