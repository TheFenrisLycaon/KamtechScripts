1. Ortho command can be operated by pressing F9.
a) True
b) False
b
2. Select the incorrect statement about Snap from the following options.
a) It is the smallest invisible distance of increment that can be set for the mouse
b) If it is set ON, the mouse moves in steps of the set increment
c) If it is set OFF, the mouse cannot move to any intermediate point between the dots of the grid
d) Function key F9 is used to make Snap on/off
c
3. If _______ mode is set on then all lines are drawn along the X and Y direction only.
a) snap
b) osnap
c) ortho
d) grid
c
4. Function key F9 is used to toggle _______
a) snap
b) ortho
c) osnap
d) grid
a
5. Which function key is used to set Grid on or off?
a) F7
b) F4
c) F3
d) F5
a
6. Symbol for quadrant snap, which appears on the entity while selecting is ____________
a) □
b) ◊
c) ○
d) ×
b
7. It is difficult to pick a point exactly and quickly with the mouse. In order to allow specifying a point exactly, which of the following command will help?
a) Grid
b) Ortho
c) Snap
d) OSNAP
d
8. Object snap increases ________
a) speed, accuracy and ease of drafting
b) speed of drawing
c) memory utilization
d) accuracy while drafting
a
9. Which function key is used to make OSNAP on/off?
a) F5
b) F3
c) F4
d) F7
b
