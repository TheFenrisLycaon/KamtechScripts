1. The drawing in which the view is drawn at an angle 30° and 150° is called ___________
a) Auxiliary view
b) Front view
c) Isometric view
d) Perspective view
c
2. The view in which the victory stand is drawn is known as __________
a) auxiliary view
b) 3D view
c) perspective view
d) isometric view
d
3 All horizontal lines are in isometric drawing is _________
a) parallel to OX
b) parallel to OZ
c) parallel to OY
d) inclined to OX
a
4. Angles of inclined surfaces are the actual angles in isometric views.
a) True
b) False
b
5. The scale which is used in the isometric drawing is known as ________
a) simple scale
b) reducing scale
c) enlarged scale
d) isometric scale
d
6. The command which is used for drawing isometric grid is _____
a) XLINE
b) POLYLINE
c) DDRMODES
d) CIRCLE
c
7. The default spacing in GRID Y is _________ mm.
a) 11
b) 21
c) 12
d) 10
d
8. The cursor does not change when we ON the isometric grid in Auto cad.
a) True
b) False
a
