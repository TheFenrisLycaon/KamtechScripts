1. It is a box section having a top cover. For which of the
following part of lathe, above sentence is true?

b) all gear drive

b

a) removable
b) non-removable
c) can’t say anything
d) none of the mentioned
a

b) a brake
c) clutches and a brake both
d) none of the mentioned
c

a) true
b) false
b

b) false

b) only two
