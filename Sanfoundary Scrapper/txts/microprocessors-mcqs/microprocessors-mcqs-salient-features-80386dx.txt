1. The 80386DX is a processor that supports
a) 8-bit data operand
b) 16-bit data operand

d) all of the mentioned

a) 8 address lines
b) 16 address lines
c) 32 address lines

c

a) 2

c) 8
d) 16
c

a) virtual memory
b) paging
c) four levels of protection
d) all of the mentioned
d

a) segmentation
b) paging
c) memory division
d) none of the mentioned
b

a) 64 pin
b) 128 pin

d) 142 pin
c

a) 12 MHz and 20 MHz
