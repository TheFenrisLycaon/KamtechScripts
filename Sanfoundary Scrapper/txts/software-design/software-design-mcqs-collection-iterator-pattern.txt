b) List

d) All of the mentioned

d) None of the mentioned

a) Initialize
b) Information Hiding
c) Multiple iterations
d) All of the mentioned
a
5. Which of these are necessary requirements for Iteration mechanism?
a) Initialize
b) Completion Test
c) Information Hiding
d) Access Current

a) initialize
b) Information Hiding
c) Advance current Test
d) Flexibility

b) External
