1. A ____________ loop is a loop whose processing is controlled by a counter.
a) Counter-controlled
b) Entry-controlled
c) Exit-controlled
d) Accumulator-controlled
a
2. ____________________ is a counter-controlled loop.
a) For..next loop
b) Do..while loop
c) While loop
d) If statement
a
